# Sales_performance_data_analysis-python-project
Data analytics project using Python , Pandas and Matplotlib

# Sales Performance Analysis
## Project Overview
This project analyzes retail sales data to understand sales trends, category performance,
and profit insights using Python and data visualization techniques.

## Tools & Technologies
- Google Colab
- Python
- Pandas
- NumPy
- Matplotlib
- Excel (for initial data cleaning)
- Power BI (for dashboarding)

## Dataset
Sample Superstore Dataset (CSV format)

## Workflow
1. Uploaded CSV file into Google Colab
2. Cleaned data (missing values, duplicates)
3. Performed exploratory data analysis using Pandas & NumPy
4. Visualized category-wise sales using Matplotlib
5. Prepared cleaned data for Power BI dashboard

## Key Insights
- Technology category generated the highest sales
- Furniture category showed lower profit margins
- Average order value was calculated for business understanding

## Conclusion
This project demonstrates an end-to-end data analytics workflow from raw data to insights
and dashboard-ready output.
